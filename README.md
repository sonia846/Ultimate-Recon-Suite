# 🚀 Ultimate Recon & Security Suite (10-in-1)
A high-performance, automated Python framework built for comprehensive network scanning, threat intelligence gathering, and security auditing.

## ✨ Advanced Features Included (10-in-1 Suite)
1. **Reverse DNS Lookup** - Resolves target IP addresses back to their official hostnames.
2. **IP Geolocation Tracker** - Fetches real-time Country, City, Region, and ISP network details.
3. **WHOIS Domain Registry Analyst** - Queries public domain registration ownership and expiry data.
4. **VPN / Proxy Traffic Detector** - Identifies routing nodes to check if traffic is hidden.
5. **DNS Subdomain Enumerator** - Automates the discovery of hidden subdomains and web infrastructure.
6. **Multi-threaded Port Scanner** - High-speed, parallel processing engine for open port discovery.
7. **Directory Bruteforce Engine** - Scans target web servers for hidden admin panels, paths, and sensitive files.
8. **Service Banner Grabber** - Extracts raw banners to fingerprint target operating systems and server software.
9. **CMS Platform Detector** - Automatically detects if the website architecture runs on WordPress, Joomla, or Drupal.
10. **Auto-Reporting Engine** - Consolidates all 10 modules' output instantly into a formatted, structured `.txt` file.
